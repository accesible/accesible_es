📜 Manifiesto

Queremos una web más justa e igualitaria

Accesible.es es un proyecto libre sin ánimo de lucro.

Dado que la accesibilidad es un derecho y nos beneficia a todos, creamos este proyecto con el objetivo de fomentar el desarrollo de sitios web accesibles.

Queremos ser una guía útil, recopilamos información sobre las pautas de accesibilidad más importantes y las resumimos para que aplicarlas en tu proyecto sea fácil. No recogemos todos los casos que puedan surgir, así que te recomendamos consultar la documentación oficial.

Este proyecto está en fase de desarrollo, en el siguiente punto te explicamos como puedes colaborar.

Cómo colaborar#

¡Cualquier colaboración es bienvenida!

Si quieres añadir contenido puedes hacerlo desde la cuenta de github, además en la parte inferior de las páginas hay un enlace para ser directamente editadas en github en formato markdown

Para cualquier duda puede contactarnos en el correo accesible.es@gmail.com
